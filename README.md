# UAC System Capacity Analytics Dashboard

## Project Overview
This project analyzes system capacity and care load for unaccompanied children using healthcare data.

## Features
- Data cleaning & preprocessing
- KPI analysis (Total Load, Net Intake, Backlog)
- Interactive dashboard using Streamlit
- Time-series forecasting using ARIMA

## Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib
